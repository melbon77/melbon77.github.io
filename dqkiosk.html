Dairy Queen Kiosk

UI • UX • Heuristic Evaluation

Designing a self-service kiosk for DQ locations.

Overview

The goal of this design is to provide a quick, intuitive, and pleasant user experience for ordering items at Dairy Queen's (DQ's) restaurants.

Timeline - 15 weeks

Tools - Figma, Illustrator, Cardboard, Tape

Heuristic Evaluation

I tested kiosks in popular restaurants to find successful elements as well as common pitfalls.

I looked at Taco Bell, Panera Bread, McDonald's, and did a deep-dive into the kiosk at Crumbl Cookies, which you can view using the link below.

Workflow

I determined what choices a user may need to take and organized them into work flows.

This helped to determine what screens, popups, or item options would be needed in the interface.

Physical Mockup

I created a full scale physical mockup of the kiosk in order to appropriately size buttons and text in the interface.

Wireframes

Final Designs

I took style pointers from DQ's existing website and mobile app when choosing my own color palette and design language.

Since the side navigation bar is mostly needed when a user first enters the menu, it collapses into a smaller format on subsequent pages, providing more real estate for the main page content.

Prototype

This prototype takes you through a user's experience choosing and ordering items from DQ's menu.

Next Steps

With more time to put towards this design, I would like to make the idle screen more visually appealing, as well as continue the user flow through the checkout sequence.
